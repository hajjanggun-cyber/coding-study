# 📘 코딩 용어 정리 시스템 - 사용자 가이드

**버전**: 1.3  
**최종 업데이트**: 2026-02-11

---

## 1. 프로젝트 소개 (이게 뭔가요?)
이 프로젝트는 코딩 공부를 하면서 알게 된 **용어, 명령어, 유용한 사이트, 아이디어** 등을 **구글 엑셀(스프레드시트)**에 자동으로 저장하고 관리해주는 나만의 웹사이트입니다.

복잡한 서버를 돈 내고 빌릴 필요 없이, **HTML 파일 하나**와 **구글 계정**만 있으면 평생 무료로 사용할 수 있습니다. 컴퓨터(PC)와 스마트폰 어디서든 접속해서 볼 수 있습니다.

---

## 2. 주요 기능 (무엇을 할 수 있나요?)

### ✅ 10가지 주제별 관리
상단 탭을 눌러 주제별로 내용을 정리할 수 있습니다.
- **기본형**: Coding용어, 프롬프트, 참고URL, 아이디어, 손님내용, 방문예정, 물건세부
- **확장형 (접기/펼치기 가능)**: 실전과정, 강의정리, 개인자료

### ✅ 간편한 등록과 수정
- **등록**: 빈칸에 제목과 설명을 적고 `데이터 등록` 버튼만 누르면 됩니다.
- **수정**: 목록에서 `[수정]` 버튼을 누르면 페이지 이동 없이 그 자리에서 바로 고칠 수 있습니다.
- **삭제**: 필요 없는 내용은 체크박스를 선택하고 `[선택 삭제]`를 누르면 됩니다.

### ✅ 자유로운 순서 변경 (✨New!)
중요한 내용은 위로 올리고, 덜 중요한 건 아래로 내리세요.
- **⏫ 맨 위로**: 한 방에 맨 꼭대기로 보냅니다.
- **🔼 위로 / 🔽 아래로**: 한 칸씩 섬세하게 이동합니다. (연속으로 누르면 빠르게 이동!)
- **⏬ 맨 아래로**: 맨 밑바닥으로 보냅니다.
*목록의 맨 위와 맨 아래, 양쪽에 컨트롤 버튼이 있어 편리합니다.*

### ✅ 똑똑한 기능들
- **전화번호 링크**: 내용에 `010-1234-5678` 같은 번호가 있으면, 클릭 한 번으로 바로 전화를 걸 수 있습니다.
- **인터넷 주소 링크**: `https://...` 주소를 적으면 클릭 가능한 링크로 자동 변환됩니다.
- **검색/필터**: 원하는 내용만 체크해서 다른 카테고리로 이사(`[선택 이동]`) 보낼 수도 있습니다.
- **보안 접속 (🔒New!)**: 앱 접속 시 비밀번호를 입력해야 내용을 볼 수 있습니다. (기본 비밀번호: 1234)

## 3. 사용 방법 (따라해 보세요)

### 데이터 입력하기
1. 원하는 **탭(카테고리)**을 클릭합니다. (예: '아이디어')
2. **용어** 칸에 제목을 적습니다. (예: '유튜브 기획')
3. **용어 설명** 칸에 상세 내용을 적습니다.
4. **[데이터 등록]** 버튼을 클릭하면 리스트에 바로 나타납니다.

### 순서 바꾸기
1. 순서를 바꿀 항목의 **체크박스**를 1개 선택합니다.
2. 위/아래 화살표 버튼(`⏫ 🔼 🔽 ⏬`)을 누릅니다.
3. 화면에서 즉시 움직이고, 자동으로 저장됩니다.
4. 다른 항목을 옮기려면 **[전체 선택]**을 두 번 눌러서 체크를 풀고 다시 선택하세요.

## 4. 설치 및 만들기 (개발자가 아니어도 가능해요)
이 프로그램은 **'눈에 보이는 화면(HTML)'**과 **'데이터 저장소(Google Sheets)'** 두 개로 이루어져 있습니다.

### 1단계: 저장소 만들기 (Google Sheets)
1. 구글 스프레드시트를 새로 하나 만듭니다.
2. 메뉴에서 **확장 프로그램** > **Apps Script**를 클릭합니다.
3. `google-apps-script-참고.js` 파일의 내용을 전부 복사해서 붙여넣습니다.
4. **(선택)** 코드 상단의 `APP_PASSWORD = "1234"` 부분을 원하는 비밀번호로 변경합니다.
5. 우측 상단 **[배포]** > **[새 배포]**를 누릅니다.
   - 유형: **웹 앱**
   - 액세스 권한: **모든 사용자** (중요!)
6. 배포 후 나오는 **웹 앱 URL**을 복사해둡니다.

### 2단계: 내 컴퓨터에 설치하기 (HTML)
1. `index.html` 파일을 메모장(또는 VS Code)으로 엽니다.
2. 코드 위쪽에 있는 `const API_URL = "..."` 부분을 찾습니다.
3. 아까 복사한 **웹 앱 URL**을 따옴표 안에 붙여넣고 저장합니다.
4. 이제 `index.html` 파일을 더블 클릭하면 바로 실행됩니다!

## 5. 개발자를 위한 요약 (심화)
이 코드가 어떻게 작동하는지 궁금한 분들을 위한 기술 요약입니다.

*   **Frontend**: Vanilla JS, HTML5, CSS3 (No Framework).
    *   **Optimistic UI**: 서버 응답을 기다리지 않고 화면(DOM)을 먼저 갱신하여 앱처럼 빠른 반응속도를 제공합니다.
    *   **Responsive**: CSS Grid와 Flexbox를 사용하여 모바일/데스크톱에 자동으로 반응합니다.
*   **Backend**: Google Apps Script (Serverless).
    *   `doGet/doPost`를 통해 JSON 데이터를 주고받습니다.
    *   `reorder` 기능은 `sheet.moveRows()` API를 사용하여 실제 엑셀 행을 이동시킵니다.
    *   **Security**: `APP_PASSWORD` 검증 로직이 추가되었습니다.
*   **Database**: Google Sheets.
    *   10개의 시트가 자동으로 생성되고 관리됩니다.
    *   각 행은 고유 ID를 가집니다.
